# Технологический Стек и Зависимости (TECH-STACK.md)

## Основной стек
- **Frontend:** React 18, Next.js (App Router), TailwindCSS
- **Backend:** Node.js, Express / Next.js API Routes
- **Database:** PostgreSQL (via Prisma ORM / Drizzle)
- **Authentication:** Supabase Auth / NextAuth

## Команды запуска (Scripts)
- `npm run dev`: Запуск локального сервера разработки.
- `npm run build`: Сборка проекта для продакшена.
- `npm run start`: Запуск собранного проекта.
- `npm run test`: Запуск unit-тестов (Jest/Vitest).
- `npm run lint`: Проверка кода линтером.

## Зависимости (Key Dependencies)
- `lucide-react`: Иконки
- `zod`: Валидация схем
- `axios` / `fetch`: HTTP запросы
