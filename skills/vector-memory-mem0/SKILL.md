# Векторная Память (ChromaDB Local)

В ходе глубокого аудита проекта подтверждено, что в экосистеме УЖЕ развернута векторная память!
Оркестратор и подагенты обязаны использовать существующий скрипт `C:\Projects\ultracode-plugin\vector-memory\rag_orchestrator.py`.

## Локация базы данных
- **Движок:** ChromaDB
- **Путь к базе:** `E:\Ultracode_VectorDB`

## Механика использования
При старте новой сессии агент должен вызывать `rag_orchestrator.py --action search --query "..."` для извлечения долгосрочного контекста. Для сохранения новых знаний используется флаг `--action ingest --file "..."`. Эта система заменяет любые внешние облачные решения (типа Mem0), обеспечивая 100% приватность и хранение на вашем выделенном жестком диске.
