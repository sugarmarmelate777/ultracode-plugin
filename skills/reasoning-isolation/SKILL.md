# CoT Isolation (o1/R1 Style)

Агенты должны использовать внутренние рассуждения, но не засорять ими основной лог.
1. Записывать все `<thought>` блоки в локальный кэш-файл `.ultracode/reasoning_cache.md`.
2. Возвращать родительскому агенту только итоговый код или прямой ответ.
3. Изоляция мыслей защищает контекстное окно главного оркестратора от переполнения.
