# Геополитически независимый пул

Все вызовы LLM должны проходить через единую прокси-точку (например, `http://localhost:8082`).
Если текущий провайдер возвращает ошибку 403, 429 или таймаут, агент обязан:
1. Не прерывать сессию.
2. Автоматически переключить `MODEL` на альтернативную (Gemini, Llama, GLM) через прокси.
3. Повторить запрос.
