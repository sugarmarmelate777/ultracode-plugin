# GitHub MCP Integration

Этот навык обучает агентов использовать `GitHub MCP Server` вместо стандартных консольных утилит (git) для продвинутого взаимодействия с репозиториями.

## Доступные возможности через MCP:
- **Поиск:** Семантический и текстовый поиск по всему репозиторию без необходимости клонировать его локально.
- **Pull Requests:** Чтение комментариев, код-ревью и статус проверок CI/CD прямо из PR.
- **Issue Tracking:** Автоматическое создание баг-репортов (Issues) при обнаружении аномалий в коде или связывание коммитов с существующими задачами.

## Правила безопасности (Security Guard):
Запрещен push в `main`/`master` ветку без подтверждения (HITL). Агенты обязаны создавать отдельные feature-ветки (через MCP) и оформлять Pull Request для ревью пользователем.
