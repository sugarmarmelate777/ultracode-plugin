# Frattini Subgroup Computation

### Definition

The Frattini subgroup \(\Phi(G)\) of a finite group \(G\) is the intersection of all maximal subgroups of \(G\); it is characteristic and consists of the set of non‑generators.

### Practical Algorithm (for groups of moderate size)

1. **Enumerate all subgroups** of \(G\) (e.g., via cyclic extension or using a subgroup lattice from a library).
2. **Identify maximal subgroups:** a proper subgroup \(M\) is maximal if there is no subgroup \(H\) with \(M < H < G\).
3. **Compute intersection:** \(\Phi(G) = \bigcap_{M \text{ maximal}} M\). If no maximal subgroups exist (i.e., \(G\) is trivial), \(\Phi(G)=G\).

### Alternative characterizations for speed

- For a nilpotent group (especially a \(p\)-group), \(\Phi(G) = [G,G]\,G^p\) (product of the commutator subgroup and the subgroup generated by \(p\)-th powers). Check nilpotency first.
- In general, \(\Phi(G)\) can also be obtained as the intersection of the kernels of all irreducible complex representations; this can be used if character tables are readily available.

### Example threshold

- For \(S_3\), the maximal subgroups are the three subgroups of order 2; their intersection is trivial, so \(\Phi(S_3)=\{1\}\).
- For \(Q_8\), the only maximal subgroups are the three cyclic subgroups of order 4; they intersect in the centre \(\{\pm1\}\), so \(\Phi(Q_8)=\{\pm1\}\).
