# Solvability Test for Finite Groups

### Purpose

Decide whether a finite group \(G\) is solvable. A group is solvable if its derived series \(G^{(0)} = G,\; G^{(n+1)} = [G^{(n)}, G^{(n)}]\) eventually becomes the identity.

### Algorithm

1. **Input:** Finite group \(G\) in a concrete form.
2. **Initialise:** \(H = G\), \(n = 0\).
3. **Repeat:**
   - Compute the derived subgroup \(K = [H, H]\), the subgroup generated by commutators of elements of \(H\).
   - If \(|K| = |H|\), stop.
   - If \(|K| = 1\), output **solvable**; the derived length is \(n+1\).
   - Else set \(H \leftarrow K\), \(n \leftarrow n+1\).
4. **Decision:** If the loop stopped because \(K = H\) and \(H \neq 1\), group is **not solvable**.

### Tools and Remarks

- The derived series of a finite group is strictly decreasing until it stabilises, so termination is guaranteed.
- Use an efficient commutator subgroup routine. For permutation groups, Schreier‑Sims methods can be employed.
- Real‑world threshold: every group of odd order is solvable (Feit–Thompson), but this algorithm provides a direct computational test for all finite groups.
- Example: \(S_4\): derived series \(S_4 > A_4 > V_4 > 1\) → solvable of length 3. \(A_5\): derived series \(A_5 > A_5\) → not solvable.
