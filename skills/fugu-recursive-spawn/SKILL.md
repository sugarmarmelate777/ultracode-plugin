# Рекурсивный спавн агентов

Оркестратор может делегировать задачу агенту-координатору.
Если координатор обнаруживает, что ему нужно обработать N независимых файлов, он обязан:
1. Создать N легковесных подзадач.
2. Вызвать инструмент маршрутизации для спавна N клонов самого себя.
3. Дождаться параллельного завершения и собрать результат.
