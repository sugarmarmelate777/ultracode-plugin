# Telegram Rich UI Bot Protocol

**Type**: `Integration / Product Feature`
**Trigger**: Разработка Telegram-ботов, автоматизация общения, чат-интерфейсы.

## Requirements
Современные боты больше не ограничиваются текстовым вводом/выводом. Агент должен использовать возможности Telegram Bot API 10.1+.

## Architecture Guidelines
1. **Rich UI (Web Apps):**
   Вместо сложных Inline-клавиатур используйте Telegram Web Apps (Mini Apps) для форм ввода, дашбордов и сложных взаимодействий. Агент должен генерировать фронтенд (React/HTML) специально под `window.Telegram.WebApp`.
2. **Text Streaming:**
   Для генерации длинных ответов от LLM используйте механизм редактирования сообщений для создания эффекта "печатания" (streaming).
   - *Совет:* Обновляйте сообщение не при каждом новом символе (лимит API), а батчами (например, каждые 1-2 секунды).
3. **Contextual Commands:**
   Используйте динамическое меню команд, которое меняется в зависимости от состояния пользователя (например, меню "Админ" vs "Пользователь").

## Execution
Когда пользователь заказывает "сделать Telegram бота", агент обязан предложить использование Web App для интерфейса и интеграцию LLM со стриминговым выводом.
