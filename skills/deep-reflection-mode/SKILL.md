# Deep Reflection Mode (Ultra Sync)

**Type**: `Context Budget / Cognitive Override`
**Trigger Words**: `/deep-think`, `ultra sync`, `максимальный фокус`, `deep reflection`

## Protocol
Обычно агенты `Ultracode` подчиняются `token-economy` и `context-budget-manager` для минимизации затрат. Однако, когда пользователь активирует `Deep Reflection Mode`, агент получает право на **полный вычислительный бюджет**.

### Behavioral Changes in Deep Reflection:
1. **Extended CoT (Chain of Thought):**
   Агент должен потратить максимальное количество токенов в блоке `<thinking>`, расписывая каждую альтернативу архитектурного решения.
2. **Self-Critique:**
   Агент должен явно оспаривать собственные решения минимум 2 раза перед их утверждением. 
3. **No Corners Cut:**
   Запрещены ленивые ответы. Код генерируется целиком (никаких `// ... остальной код`), даже если это занимает тысячи строк.
4. **Multi-Step Verification:**
   Вместо того чтобы сразу писать код, агент сперва создает артефакт `implementation_plan.md` с глубоким архитектурным анализом.

## Usage
Этот режим следует применять только для фундаментальных архитектурных изменений, написания сложных алгоритмов или когда предыдущие быстрые попытки завершились неудачей. При активации агент должен явно сообщить пользователю: *"Deep Reflection Mode активирован. Бюджет токенов расширен. Начинаю глубинный анализ."*
