# Fresh Docs Context7 Protocol

**Type**: `Knowledge Base Enhancer`
**Integration**: `mcp-context7` (MCP Server)

## Context Window Limitations
Языковые модели ограничены своей датой обучения. Если пользователь просит использовать новейшие фичи (например, React 19 Server Actions, Next.js 15, Drizzle ORM latest), высок риск галлюцинаций. 

## Directive
**ПЕРЕД** тем как писать код, использующий новейшие библиотеки или фреймворки, агент **ОБЯЗАН** обратиться к MCP серверу `context7`, чтобы получить самую актуальную документацию.

## Protocol Steps
1. Определить стек технологий из запроса пользователя.
2. Использовать lazy-loaded tool `call_mcp_tool` для сервера `context7`.
3. Найти нужную библиотеку через `resolve-library-id` (например, `react`, `nextjs`).
4. Сделать запрос `query-docs` по конкретной фиче (например, "React 19 useFormStatus").
5. Прочитать полученную документацию и только после этого приступать к кодингу.

Это гарантирует, что генерируемый код будет соответствовать последним стандартам, а не устаревшим паттернам из тренировочной базы модели.
