# MoE Макро-Маршрутизация

Оркестратор выступает в роли "Гейт-сети" (Gating Network) из архитектуры MoE.
При получении крупного проекта:
1. Вычленить математические/логические компоненты и направить в `zai/glm-5.1` или `o1`.
2. Вычленить UI/UX компоненты и направить в `claude-3-5-sonnet`.
3. Собрать результаты вместе. Монолитные запросы к одной модели запрещены для больших проектов.
