# Omni Profiler Swarm

## Directives

1. **Five-Dimensional Profiling Architecture:**
   - **Psych Profiler Agent**: Analyzes text corpus (emails, posts, messages) for personality traits using OCEAN (Big Five) model, linguistic markers (LIWC-style), sentiment progression, and cognitive bias patterns.
   - **Financial Profiler Agent**: Maps income streams, spending patterns, asset ownership, debt structure, and financial risk tolerance from transaction data, tax filings, and public records.
   - **Social-Network Profiler Agent**: Constructs relationship graphs, identifies key influencers, maps group affiliations, measures social capital, and detects anomalous connections.
   - **Temporal-Behavioral Profiler Agent**: Builds activity chronologies, detects routine deviations, identifies periodicity patterns, and correlates behaviors across time zones.
   - **Geospatial Profiler Agent**: Maps location history, identifies frequently-visited locations, detects travel patterns, correlates location with other profile dimensions.

2. **Swarm Convergence Protocol:**
   - Each agent produces a dimension-specific profile with confidence scores per finding.
   - Magic Deduction Swarm integration: profiles with conflicting findings trigger adversarial debate.
   - Unified dossier generated via structured-swarm-artifacts JSON protocol.
   - Confidence-weighted composite score for each profile assertion.
   - Dissenting agent positions preserved in minority report appendix.

3. **Data Source Integration:**
   - Ingests from osint-detectives (social media, public records).
   - Ingests from browser-use-agent (interactive data gathering).
   - Ingests from web-devourer (bulk corpus ingestion).
   - Ingests from mcp-business-integrations (structured business data).
   - Ingests from forensic-audio-transcriber (audio-derived behavioral markers).

4. **Governance & Compliance:**
   - All profiling subject to immutable-core-directives: no profiling without explicit lawful purpose.
   - security-guard: no unauthorized data access. Only public or legally-authorized private data.
   - cryptographic-integrity-monitor: every source datum hashed and chained for auditability.
   - GDPR/CCPA compliance: right-to-know summaries generated for any profiled subject.
   - Profile data classified CRITICAL — never persisted beyond session without explicit user approval.

5. **Output Artifacts:**
   - Unified Subject Dossier (JSON via structured-swarm-artifacts).
   - Per-Dimension Profile Reports with confidence scoring.
   - Relationship Graph (nodes, edges, weights) in structured format.
   - Risk Indicator Matrix: cross-dimensional risk scoring.
   - Minority Reports: any dissenting agent findings preserved.

6. **Integration Map:**
   - **Upstream**: osint-detectives, browser-use-agent, web-devourer, mcp-business-integrations, forensic-audio-transcriber.
   - **Downstream**: magic-deduction-swarm, legal-expert-swarm, clinical-diagnostician, finance-accounting-agent.
   - **Governing**: security-guard, immutable-core-directives, cryptographic-integrity-monitor, zero-trust-orchestration.
   - **Council**: EARTH (Structure & Crystal domain).
