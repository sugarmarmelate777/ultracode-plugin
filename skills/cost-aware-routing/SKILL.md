# Cost-Aware Routing (Экономное Делегирование)

В инфраструктуре `Ultracode Dual-Core` установлены следующие тиры маршрутизации для `DeepSeekEngine`:
- **Haiku (gemini-3.1-flash-lite)** — сверхбыстрые и легкие задачи, базовая обработка текста (бесплатно).
- **Sonnet (nvidia_nim/nemotron-4-340b-instruct)** — основная рабочая лошадка для генерации кода и фронтенда (бесплатно).
- **Default (nvidia_nim/llama-3-70b-instruct)** — стандартный резерв (бесплатно).
- **Opus (deepseek-v4-pro)** — платная модель с максимальным IQ.

## Правила делегирования:
Оркестратор ОБЯЗАН экономить бюджет пользователя (Free-Tier First). 
1. По умолчанию для 90% задач, связанных с UI, версткой, рефакторингом, и поиском информации, подагентам запрещено требовать модель уровня `Opus` или платный `DeepSeek`. Направляйте их на уровень `Sonnet` или `Default`.
2. Платную модель (`Opus` / `DeepSeek-V4-Pro`) разрешено запрашивать ТОЛЬКО в случае:
   - Тяжелой математики.
   - Разработки сложной серверной бизнес-логики.
   - Решения критических багов (Code-Red), где легкая модель уже не справилась.
