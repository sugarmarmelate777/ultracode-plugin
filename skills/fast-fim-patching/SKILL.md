# Fast Fill-in-the-Middle Patching

Если задача требует изменения одной или нескольких строк без переписывания всего файла:
1. Запрещено использовать тяжелые модели (Claude Opus, GPT-5).
2. Маршрутизация идет к быстрым моделям (Mistral/Llama 70B) с поддержкой FIM (Fill-in-the-Middle).
3. Передается только префикс кода, суффикс и инструкция по заполнению прорехи.
